📚 Quick Reference

🎯 What is this project?

A professional, responsive hospital website built with HTML, CSS, and JavaScript using component-based architecture.

🚀 How do I start?

Open index.html in your browser to see the website. Read QUICK_START.html for a guided tour.

🔧 How do I customize?

Edit component files in the components/ folder to change content, or edit css/styles.css for styling.

📦 How do I deploy?

See DEPLOYMENT_GUIDE.md for instructions on deploying to GitHub Pages, Netlify, Vercel, or traditional hosting.

📱 Is it responsive?

Yes! The website is fully responsive with breakpoints for mobile, tablet, and desktop screens.

🎨 Can I change colors?

Yes! Edit the CSS variables in css/styles.css to change the entire color scheme.
